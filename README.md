**运行说明**

选择一个版本文件夹（如*beta-260515*），根据需要下载文件夹内的内容

    main.cpp ：游戏c++源代码（GB-2312编码）

    bgame.h ：编译游戏源代码所必需的头文件（UTF-8编码）

    main.exe ：编译完成的游戏程序（可在Windows各版本运行）

注意：游戏程序的窗口大小不可更改，在运行时可能会出现一些显示问题，将于之后逐步解决完善，敬请谅解！

**编译说明**

编译游戏源代码前需要正确安装了**EasyX库**的c++环境（EasyX库官网下载：https://easyx.cn/easyx）

本项目*main.cpp*使用GB-2312编码，*bgame.h*使用UTF-8编码。至少使用c++11标准和8.1.0及以上的g++.exe

参考g++版本

    g++.exe (x86_64-posix-seh-rev0, Built by MinGW-W64 project) 8.1.0

参考编译命令（不同版本某些选项可能不需要，文件名及路径请自行修改）

    g++.exe -std=c++11 -static -static-libgcc -static-libstdc++ -pthread main.cpp -o main.exe -leasyx -Wl,--whole-archive -lpthread -Wl,--no-whole-archive
