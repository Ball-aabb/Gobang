**作者：Wzh**

**联系邮箱：wzh_2011@139.com**

\--------------------------------------------

**使用说明**

选择一个版本文件夹（如beta-251006），根据需要下载文件夹内的内容

    main.cpp ：游戏c++源代码

    bgame.h ：编译游戏源代码所必需的头文件

    main.exe ：编译完成的游戏程序

**编译说明**

编译游戏源代码前需要：
    
    安装c++环境（自行安装）
    
    安装EasyX库（官网下载链接：https://easyx.cn/easyx）

**beta版说明**

目前游戏处于开发阶段（beta版本），可能会存在系统兼容性差、不稳定、界面混乱、程序出错等各种原因不明的问题，敬请谅解
