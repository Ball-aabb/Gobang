**使用说明**

选择一个版本文件夹（如beta-250914），根据需要下载文件夹内的内容

    main.cpp ：游戏c++源代码

    bgame.h ：编译游戏源代码所必需的头文件

    main.exe ：编译完成的游戏程序

**编译说明**

编译游戏源代码前需要：
    
    安装c++环境（自行安装）
    
    安装EasyX库

EasyX库官网下载链接：https://easyx.cn/easyx
